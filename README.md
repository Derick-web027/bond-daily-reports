# 固收市场日报 Bond Daily Reports

买方信用策略分析师视角，基于Wind数据自动生成的日度固收市场复盘。

🌐 **在线查看：** https://derick-web027.github.io/bond-daily-reports/

## 内容结构

每期日报包含三大板块：
- **资金面** — 央行操作、DR001/DR007/R007、CNEX资金情绪、展望
- **利率债** — 曲线形态、关键活跃券、期限/品种利差、驱动分析、一级市场
- **信用债（核心）** — 整体定性、信用利差、二永、城投、产业、策略建议

## 输出形式

- 📄 **Word文档**（详细版+精简版）：本地归档
- 🌐 **HTML网页**（本仓库）：随时在线查看，支持精简/详细版切换

## 更新频率

每工作日18:00后（基于Wind债市综述PDF+活跃券图+Wind EDB API）。
